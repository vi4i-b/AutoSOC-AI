# Материал для презентации AutoSOC AI

## Название проекта

AutoSOC AI

## Короткий pitch

AutoSOC AI — это настольный помощник по кибербезопасности, который помогает небольшим командам находить опасные порты, блокировать риски и получать мгновенные уведомления в Telegram.

## Проблема

У маленьких компаний, студенческих команд и локальных администраторов часто нет:

- полноценной SOC-команды
- дорогих систем безопасности
- понятных инструментов для неэкспертов

Из-за этого:

- опасные порты остаются открытыми
- пользователи не понимают уровень риска
- реакция на угрозы запаздывает

## Решение

AutoSOC AI объединяет в одном приложении:

- сканирование портов
- анализ риска
- управление firewall
- AI-пояснения
- Telegram alert

Это делает кибербезопасность более доступной и понятной.

## Основная ценность проекта

- простой пользовательский сценарий
- быстрый результат
- возможность не только увидеть проблему, но и сразу отреагировать
- знакомый канал уведомлений через Telegram
- AI, который объясняет риски простым языком

## Основной функционал

## 1. Вход и регистрация

- login и registration окно
- привязка Telegram Chat ID к аккаунту
- ограничение `1 Telegram Chat ID = 1 аккаунт`

## 2. Telegram интеграция

- пользователь пишет `/start`
- бот возвращает `Telegram User ID` и `Telegram Chat ID`
- пользователь вставляет Chat ID при регистрации
- результаты сканов и alert уходят в этот чат

## 3. Сканирование портов

- проверка отслеживаемых TCP-портов через Nmap
- вывод checked/open/closed/filtered
- отображение открытых сервисов

## 4. Анализ риска

- выявление опасных сервисов, например SMB, RDP, FTP, Telnet
- подсчёт risk score
- выделение рискованных портов

## 5. Активное реагирование

- пользователь может открывать и закрывать порты из интерфейса
- применяются правила Windows firewall
- dashboard обновляется в реальном времени

## 6. AI-помощник

- объясняет результаты скана
- подсказывает следующие шаги
- снижает порог входа для неэкспертов

## Почему проект сильный для хакатона

- решает понятную реальную проблему
- сочетает cybersecurity, UI, automation, AI и messaging
- хорошо демонстрируется вживую
- показывает практическую пользу, а не только идею

## Сценарий демо

1. Открыть login/registration окно
2. Показать Telegram-бота
3. Отправить `/start`
4. Показать получение Chat ID
5. Создать аккаунт с этим Chat ID
6. Войти в приложение
7. Запустить скан
8. Показать dashboard, risky ports и AI summary
9. Показать отправку результата в Telegram
10. Заблокировать порты и показать live-обновление панели

## Инновационные моменты

- Telegram-связка прямо в процессе регистрации
- real-time dashboard по exposure портов

- AI-слой поверх кибербезопасности
- упрощение security workflow для неэкспертов

## Техническая архитектура

- UI: CustomTkinter
- Scan engine: Nmap
- Monitoring: Scapy-based guard
- Storage: SQLite
- Notifications: Telegram Bot API
- Packaging: PyInstaller
